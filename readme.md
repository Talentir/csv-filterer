# Install Instructions
## Install NVM
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```
```bash
touch ~/.zshrc
```
```bash
nvm use 18
```

## Install code
```bash
cd ~/Download/csv-filterer-main
```
```bash
npm install
```

## Adjust file names
Edit `src/index.ts`


## Run code
```bash
npm run filter
```
